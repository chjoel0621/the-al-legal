# 계정 및 데이터 삭제 안내

**디알(The-al)** 은 이용자가 언제든지 계정과 모든 관련 데이터를 삭제할 수 있도록 지원합니다.

---

## 1. 앱에서 직접 삭제하기 (즉시)

가장 빠른 방법입니다.

1. 디알 앱 실행 → **마이페이지** 이동
2. 하단의 **회원 탈퇴** 선택
3. 안내에 따라 탈퇴 진행

탈퇴 시 계정과 모든 데이터가 **즉시 삭제**되며, 복구할 수 없습니다.

---

## 1-1. 계정을 유지하면서 일부 데이터만 삭제하기

계정을 삭제하지 않고도, 앱 안에서 개별 데이터를 직접 삭제할 수 있습니다.

- **영양제·의약품 삭제:** 홈 또는 상세 화면에서 해당 항목을 삭제
- **복용 기록 삭제:** 복약 체크를 해제하여 기록 제거
- **가족 구성원 삭제:** 가족 화면에서 멤버 삭제 (해당 멤버의 영양제·복용 기록도 함께 삭제됨)
- **건강 정보 수정·삭제:** 마이페이지에서 건강 고민·만성 질환 등 수정 또는 비우기
- **돌봄 연결 해제:** 연결된 가족과의 돌봄 연결 해제

직접 처리가 어려운 경우 아래 이메일로 "삭제를 원하는 데이터 항목"과 가입 이메일을 알려주시면 처리해 드립니다.

## 3. 삭제되는 데이터

회원 탈퇴 시 다음 데이터가 **완전히 삭제**됩니다.

- 계정 정보 (이메일, 별명)
- 건강 프로필 (출생연도, 성별, 건강 고민, 만성 질환)
- 등록한 영양제·의약품 정보 및 복용 기록
- 가족 구성원 및 돌봄 연결 정보
- 푸시 알림 토큰

데이터베이스에서 즉시 삭제되며, 자동 백업본은 백업 보존주기(최대 7일)에 따라 자연 소멸합니다. 삭제된 데이터는 복구되지 않습니다.
